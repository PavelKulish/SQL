# Репозиторий учебных проектов по SQL

Данный репозиторий содержит мои работы, выполненные в ходе прохождения курса по SQL от **Лаборатории НИКС**. Здесь представлены как небольшие тренировочные задания, так и более сложные проекты, демонстрирующие применение SQL для решения практических задач.

## Структура репозитория

### **Small Trainings (`/small_trainings`)**
Серия небольших домашних заданий, сфокусированных на отработке ключевых аспектов SQL.

#### **SHW 1: Базовый синтаксис**
- Ввод данных, фильтрация и сортировка
- Использование функций для работы с датами

#### **SHW 2: Группировки и агрегатные функции**
- Применение оператора `GROUP BY`
- Использование агрегатных функций (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)

#### **SHW 3: Соединения таблиц и работа с NULL**
- Практика использования `INNER JOIN` и `LEFT JOIN`
- Обработка значений `NULL` в запросах

#### **SHW 4: Подзапросы и операции с множествами**
- Написание коррелированных и некоррелированных подзапросов
- Использование предикатов `EXISTS`, `ANY`, `ALL`
- Применение операторов `UNION` и `EXCEPT` для работы с наборами данных

#### **SHW 5: Операции DML и DDL**
- Создание и изменение структур таблиц (`CREATE`, `ALTER`, `DROP`)
- Манипуляция данными: вставка, обновление, удаление записей (`INSERT`, `UPDATE`, `DELETE`)

#### **SHW 6-7: Работа с графами**
- Представление графов в реляционной модели
- Построение полных (транзитивных) графов
- Определение состава изделия с учётом иерархии компонентов
- Поиск критических путей в сетевых графах

#### **SHW 8: Работа с текстом и продвинутые конструкции**
- Рекурсивные CTE для генерации последовательностей
- Парсинг и анализ текстовых строк
- Создание пользовательских функций и процедур
- Применение `CROSS APPLY` и `OUTER APPLY`

#### **SHW 9: Триггеры и логирование**
- Создание триггеров для отслеживания изменений данных
- Ведение журнала операций (INSERT, UPDATE, DELETE)
- Открутка состояния таблицы на заданный момент времени (сверху и снизу)

### **Projects (`/project1`, `/project2`, `/project3`)**
Комплексные проекты, объединяющие различные навыки для решения более масштабных задач.

#### **Проект 1: Система работы с полиномами**
**Задача:** Разработать реляционную схему для хранения действительных полиномов одной переменной.

**Функционал:**
- Реализация SQL-запросов для выполнения арифметических операций над полиномами (сложение, умножение, деление)
- Решение линейных и квадратных уравнений на основе данных из БД

#### **Проект 2: Аналитическая и практическая задачи**

**Часть 1: Аналитическое исследование (ad-hoc анализ)**
- **Задача:** Рассчитать среднюю стоимость товара, перевозимого транспортными средствами классов "минивэн" и "небольшой грузовик"

**Часть 2: Практическая работа с базой данных книжного магазина**
- **Задача:** Работа с комплексной схемой БД, моделирующей работу книжного магазина
- **Включает:** анализ заказов, клиентов, склада и формирование отчетов

#### **Проект 3: Графы и геоаналитика**

**Часть 1: Аналитическое исследование**
- **Задача:** Оценка количества гипермаркетов и маленьких продуктовых магазинов на территории Москвы
- **Методы:** Работа с открытыми данными, агрегация и анализ

**Часть 2: Практическая работа с графами**
- **Задача:** Применение алгоритмов на графах для решения прикладных задач
- **Включает:** построение иерархий, расчёт путей, анализ связей

## Навыки и Технологии

- **Язык:** SQL (стандарт ANSI, с учетом особенностей MS SQL Server)
- **Ключевые концепции:** DDL, DML, подзапросы, JOIN, агрегатные функции, оконные функции, индексы
- **Продвинутые темы:** рекурсивные CTE, графы в SQL, триггеры, логирование, пользовательские функции и процедуры
- **Процесс:** Написание читаемого и эффективного кода, тестирование запросов, анализ производительности
